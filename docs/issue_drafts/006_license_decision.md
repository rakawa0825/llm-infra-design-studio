# Decide License Before Public Release

## Summary

Prepare a license decision before any public release.

## Why This Matters

Publishing without a license creates ambiguity about reuse. The decision should be explicit before the repository becomes public.

## Scope

Prepare decision options:

- No license for private preview.
- MIT.
- Apache-2.0.
- Custom / all rights reserved.

## Out of Scope

- Choosing a license in this issue draft.
- Legal advice.
- Public release.

## Acceptance Criteria

- License options are documented.
- Tradeoffs are summarized.
- Final decision remains human-owned.

## Validation

Manual review required.

## Labels

license, public-release, human-approval

## Priority

High before public release
