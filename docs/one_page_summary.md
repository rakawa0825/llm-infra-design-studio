# One Page Summary

LLM Infra Design Studio is a Markdown-first workflow template for LLM-assisted infrastructure design. It helps engineering teams convert fragmented inputs into traceable artifacts for review and approval.

The repository demonstrates a synthetic ExampleCorp scenario for branch connectivity migration to SD-WAN and cloud security. It emphasizes facts, assumptions, unresolved issues, delta analysis, and human approval gates.

This project does not automate network design. It supports engineers by making design evidence easier to inspect and reuse.
