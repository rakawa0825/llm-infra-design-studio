# Routing / SD-WAN Impact Model

## Purpose

This model defines review dimensions for routing and SD-WAN impact analysis.

## Review Dimensions

- routing domain
- underlay / overlay
- preferred path
- backup path
- local internet breakout / DIA
- cloud security steering
- failover behavior
- routing dependency
- human approval requirement

## Boundary

The model reviews routing assumptions and impact. It does not define vendor-specific configuration, production route policy, or final failover design.

## Human Review

Changes that affect preferred path, backup path, cloud security steering, or failover behavior require human review and approval.
