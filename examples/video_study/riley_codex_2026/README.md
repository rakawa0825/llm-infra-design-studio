# Riley Brown Codex 2026 Video Study

## Source

- Title: `Codex Full Course 2026: The NEW Best AI Coding Tool`
- URL: `https://www.youtube.com/watch?v=KXIdYEdOPys`
- Processing type: VTT transcript to structured study artifacts

## Purpose

This video is treated as a practical external learning resource for Codex usage.

It can be used locally as an external learning resource for Codex usage patterns, agentic workflow patterns, and OpenAI readiness preparation.

This README does not claim official OpenAI endorsement and does not include source transcript text.

## Processing Flow

1. Extract VTT with `yt-dlp`.
2. Process transcript with Codex.
3. Generate summaries and concept maps.
4. Extract workflow usage patterns.
5. Map concepts to infrastructure design lifecycle workflows.
6. Extract OpenAI readiness notes.

## Generated Artifact Types

- cleaned transcript
- segment summaries
- concept glossary
- usage patterns
- workflow mapping
- OpenAI readiness notes
- final report

## Lessons Learned

- YouTube download websites are unnecessary if `yt-dlp` works.
- Manual transcript copy-paste is inefficient.
- VTT plus Codex enables repeatable video-to-artifact processing.
- This pattern is a lightweight Field-to-Artifact pipeline.
