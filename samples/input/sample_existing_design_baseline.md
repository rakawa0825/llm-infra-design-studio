## Current Branch Traffic Assumption

Branch internet traffic is expected to use the existing branch egress model until Cloud-Security-Service inspection scope is confirmed.

## Current Monitoring Assumption

Monitoring-System receives basic SD-WAN tunnel availability events. Alert thresholds and operations ownership remain detailed-design handoff items.

## Current Communication Matrix Assumption

The communication matrix includes branch-to-cloud placeholders but does not approve all branch internet traffic steering through Cloud-Security-Service.

## Unresolved Security Inspection Scope

Inspection scope, exception behavior, and failover behavior are unresolved and require human approval before baseline update.
