# Intake

## Goal

Collect public-safe source inputs and confirm they can be used in the current cycle.

## Input

- Meeting notes
- Design excerpts
- Review comments
- Vendor answers
- Communication requirements

## Process

1. Confirm each input is synthetic or approved for public use.
2. Assign a temporary source name.
3. Identify owner and date.
4. Flag missing context.

## Output

- Intake list for source manifest creation

## Quality Gate

No confidential or private project data is accepted.

## Human Approval / Escalation

Escalate any uncertain source origin before processing.
