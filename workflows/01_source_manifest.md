# Source Manifest

## Goal

Create a traceable inventory of all accepted sources.

## Input

- Intake list
- Source metadata

## Process

1. Assign source IDs.
2. Record title, type, owner, date, and status.
3. Summarize scope.
4. Mark downstream use.

## Output

- Source manifest

## Quality Gate

Every source has an ID and sample-safety status.

## Human Approval / Escalation

Escalate missing source ownership or unclear confidentiality.
