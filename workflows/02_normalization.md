# Normalization

## Goal

Create stable terminology across sources while preserving original meaning.

## Input

- Source manifest
- Raw source text

## Process

1. Identify component names and aliases.
2. Map aliases to canonical terms.
3. Preserve source references.
4. Flag ambiguous terms.

## Output

- Normalized term map
- Ambiguity list

## Quality Gate

No ambiguity is silently resolved.

## Human Approval / Escalation

Escalate canonical naming choices that affect customer-facing artifacts.
