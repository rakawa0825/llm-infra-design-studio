# Verification

## Goal

Validate completeness, traceability, and safety of outputs.

## Input

- Sample outputs
- Templates
- Scripts

## Process

1. Check required headings.
2. Scan for sensitive identifiers.
3. Detect risky unresolved assertions.
4. Record findings.

## Output

- Verification report

## Quality Gate

No sensitive identifier or unsupported approval claim remains unresolved.

## Human Approval / Escalation

Escalate validation failures before publication.
