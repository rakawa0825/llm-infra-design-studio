# Delta Impact Analysis

## Goal

Identify impact from new or changed source information.

## Input

- Previous artifacts
- New source manifest entries
- Review comments

## Process

1. Compare old and new meaning.
2. Classify change type and impact.
3. Identify affected artifacts.
4. Propose merge, reject, or defer decision.

## Output

- Delta report

## Quality Gate

Each delta has source evidence and an explicit status.

## Human Approval / Escalation

Merge decisions require human approval.
