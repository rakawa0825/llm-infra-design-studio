# Artifact Update

## Goal

Update design artifacts after review and merge decisions.

## Input

- Delta report
- Approved requirements
- Templates

## Process

1. Select affected artifacts.
2. Apply approved updates.
3. Retain unresolved items.
4. Record change rationale.

## Output

- Updated artifact drafts

## Quality Gate

Drafts clearly distinguish approved decisions from pending items.

## Human Approval / Escalation

Escalate customer-facing text and detailed-design handoff items.
